Index

The index of a component can also be used for recognition. However, you need to differentiate between the 'Class index' of a 'Component' node and the index used in a SmartID (Preview): The first always refers to GUI elements of this 'Class name' in reference to the parent component. In the case of SmartIDs, the index refers to the eligible components for the specified SmartID (see SmartID with index).
