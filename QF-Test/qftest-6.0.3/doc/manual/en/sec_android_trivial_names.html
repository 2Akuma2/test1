Android - list of trivial component identifiers

The following component identifiers are not transferred to the 'Name' attribute by default. This can be controlled via a NameResolver (see subsection 50.1.6).

  • canvas
  • checkbox
  • choice
  • container
  • content
  • dialog
  • dock
  • drawer
  • filedlg
  • frame
  • label
  • list
  • menu
  • menubar
  • menuitem
  • pager
  • panel
  • popup
  • row
  • scrollbar
  • scrollpane
  • tabs
  • textfield
  • toolbar
  • title
  • text
  • win
