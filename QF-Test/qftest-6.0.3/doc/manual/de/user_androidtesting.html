6.0+15
Testen von Android Anwendungen

Diese Kapitel behandelt die Testautomatisierung von nativen Android Anwendungen.

Hinweis Sollten Sie mobile Web-Anwendungen testen wollen, schauen Sie sich bitte die Möglichkeiten der mobilen Browser-Emulation des Chrome Desktop-Browsers an, welche im Abschnitt 13.6 beschrieben werden. Obwohl es möglich ist, einen Webbrowser auf Android Geräten (vorausgesetzt er unterstützt die Accessibility Schnittstelle) zu nutzen, bietet die mobiles Browser-Emulation bessere Automatisierungsmöglichkeiten und weniger Aufwand für mobiles Web-Testen.
