Index

Der Index einer Komponente kann ebenfalls zur Wiedererkennung verwendet werden. Allerdings ist zwischen dem 'Index' eines 'Komponente' Knoten und dem in einer SmartID (Preview) verwendeten Index zu unterscheiden: Ersterer bezieht sich immer auf GUI-Elemente dieser 'Klasse' bezogen auf die Parent-Komponente. Bei der SmartID bezieht sich der Index auf die in Frage kommenden Komponenten mit der angegeben SmartID (siehe SmartID mit Index).
